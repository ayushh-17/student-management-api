# Student Management System API

Spring Boot backend project with:

- JWT Authentication
- Role-Based Authorization
- BCrypt Password Encryption
- Swagger Documentation
- MySQL Database
- DTO Architecture
- Validation
- Global Exception Handling

## Technologies Used

- Java
- Spring Boot
- Spring Security
- JWT
- MySQL
- Hibernate / JPA
- Swagger OpenAPI

## Features

### Authentication
- Register User
- Login User
- JWT Token Generation

### Student APIs
- Add Student
- Get All Students
- Get Student By ID
- Update Student
- Delete Student

### Roles
- ADMIN → Full Access
- USER → Read Only
